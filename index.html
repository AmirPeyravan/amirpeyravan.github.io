I'm a full stack developer with a backend-first mindset, currently studying Computer Engineering at Islamic Azad University, Kazerun Branch. I started shipping production code in 2024 and have worked continuously since — across agencies, a national university system, a regional municipality, and my own company.

My core expertise is the PHP/Laravel ecosystem: REST API design, service-oriented backend architecture, and third-party system integration. Beyond the code itself, I care about how software gets delivered — CI/CD pipelines, containerized environments, and automating the repetitive parts of a release so teams can ship faster with fewer surprises.

Linux is where I actually live. I hold an LPIC-1 certification and run production workloads on Linux daily — provisioning servers, managing services, hardening access, and debugging things at the OS level when the framework isn't the problem.

In January 2025, I founded Nexavio, a software company based in Fars Province, where I lead product architecture and client delivery end-to-end. I've since worked on platforms ranging from real-estate marketplaces and OTT streaming to university and municipal systems — different domains, same standard: clean architecture, honest scoping, and software that's still maintainable a year later.
